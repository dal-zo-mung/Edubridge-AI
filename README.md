# 🎓 Edubridge-AI

Edubridge-AI is an innovative educational platform built with **Next.js**. It leverages AI technology to create a seamless bridge between modern learning resources and students.

## 🚀 Platform Access & Live Prototype
EduBridge-AI is currently available as a functional prototype, showcasing our AI-powered career ecosystem. We invite you to explore the platform via the following links:

🌐 Frontend (User Interface): [https://edu-bridge-ai-frontend.vercel.app](https://edu-bridge-ai-frontend.vercel.app)

⚙️ Backend (API Services): [https://edubridge-ai-ui2j.onrender.com](https://edubridge-ai-ui2j.onrender.com)

## 🎯 Technical Focus & Performance
AI-Centric Demo:
This prototype is specifically optimized to demonstrate our core AI-Driven Career Guidance engine. Development efforts have been concentrated on the seamless performance of the AI Learning Roadmap, AI Mock Interview, and AI Portfolio Scanner, which represent our primary value proposition.
### [!NOTE]
While these core AI features are fully functional for the purpose of this demonstration, we are continuously iterating on additional UI/UX enhancements and platform feature releases.
## ⚠️ Note on Platform Performance
Our backend services are hosted on a cloud-based server provider. If you experience a slight delay during your initial access, please be aware that the server may take a few moments to "wake up" from an idle state due to the hosting plan’s power-saving protocols. Once active, the platform will provide full functionality for your testing and review.
